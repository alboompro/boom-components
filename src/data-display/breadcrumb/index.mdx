# Breadcrumb

O breadcrumb é utilizado para orientar o usuário no caminho que foi percorrido pelas categorias, até chegar ao seu destino atual dentro do site. Sua tradução, ao pé da letra, significa migalhas de pão,
sendo uma analogia aos contos de João e Maria, que deixavam rastros de migalhas de pão para lembrar o caminho de casa.

## Quando Utilizar

O breadcrumb deve ser utilizado quando o site possui um sistema de hierarquias, para facilitar a navegação e localização do usuário.

### Exemplos

### Uso Básico
