# Card

Uma área de conteúdo retangular simples.

## Quando utilizar

- Quando o conteúdo possa consistir em vários elementos de diversos tipos e tamanhos.

## Exemplos

### Uso Básico

Um cartão básico contendo um título e conteúdo. Suportando dois tamanhos diferentes `default` e `small`.

<Playground>
  <div>
    <Card title="Default size card" style={{ width: 300 }}>
      <p>Card content</p>
      <p>Card content</p>
    </Card>

    <Card title="Small size card" size="small">
      <p>Card content</p>
      <p>Card content</p>
      <p>Card content</p>
    </Card>

  </div>

</Playground>

### Cartão sem `Bordered`

É possível criar um cartão sem o cortorno, pode ser usado em fundos em que a borda não faz contraste.

<Playground
  style={{
    backgroundColor: "#ddd",
    margin: "-2em",
    padding: "2em",
    width: "auto"
  }}
>
  <Card title="Card title" bordered={false}>
    <p>Card content</p>
    <p>Card content</p>
  </Card>
</Playground>

### Cartão simples

Eventualmente é desejável criar um cartão somente com o conteúdo, sendo possível na seguinte maneira:

<Playground
  style={{
    backgroundColor: "#ddd",
    margin: "-2em",
    padding: "2em",
    width: "auto"
  }}
>
  <Card>
    <p>Card content</p>
  </Card>
</Playground>

### Cartão `Hoverable`

Cartão com sentença de sopreposição. Usado para quando o cartão possa ser utilizado.

<Playground>
  <Card title="Card title" hoverable>
    <p>Card content</p>
    <p>Card content</p>
  </Card>
</Playground>

### Cartão com `Cover`

É possível adicionar um ou mais `node` usando a prop "cover".

<Playground>
  <Card
    style={{ width: 240 }}
    cover={
      <img
        alt="image example"
        src="https://images.pexels.com/photos/1252983/pexels-photo-1252983.jpeg?cs=srgb&dl=pexels-lisa-1252983.jpg&fm=jpg"
      />
    }
    title="Card title with cover"
    size="small"
    hoverable
  >
    <p> Card Content </p>
  </Card>
</Playground>

### Cartão usando tabs

Mais conteúdo pode ser alocado usando `tabs`

### Mais configurações ao conteúdo

Você pode usar `Card.Meta` suportando um conteúdo ainda mais fléxivel. Como o abaixo, utilizando `title`, `description` e `avatar`:
