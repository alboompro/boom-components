# Dropdown

Um menu dropdown.

## Quando Utilizar

Quando um conteúdo ou menu precisa ser exibido em uma janela dropdown.

## Exemplos

### Uso Básico

```jsx
import Dropdown from "@alboom/boom-components/data-display/dropdown"

const MyComponent = () => {
  const dropdownContent = (
    <div>
     Fico oculto até interagirem com o botão.
    </div>
  )
  return (
    <Dropdown overlay={dropdownContent}>
      <a>Clique aqui</a>
    </Dropdown>
  )
}
```
