# Select

Selects são campos usados para que se possa ser selecionado uma opção de uma lista de forma facil, e podem conter uma label, o campo de Seleção e um retorno de errro.

## Quando utilizar
* Construir Forms com campos de Select.

## Dependências
O Select não possui nenhuma dependência;

## Exemplos

### exemplo de Select simples
```jsx
  <Select
    label="Select input"
    value={value}
    placeholder="placeholder"
    options={{value: "example1", label: "example 1"},{value: "example 2", label: "example 2"}}
    onChange={e => this.setState({
      value: e.value
    })}
  />
```
