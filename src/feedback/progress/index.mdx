# Progress

Uma barra de progresso de uma tarefa.

## Quando usar?

Quando você precisa mostrar o progresso de uma tarefa.

## Exemplos

### Uso Básico

<Playground>
{
  () => {
    const [value, setValue] = React.useState(0);

    increment = () => setValue(prev => (prev < 100) ? prev + 10 : prev);
    decrement = () => setValue(prev => (prev > 0) ? prev - 10 : prev);
    reset = () => setValue(0);

    return (
      <>
        <div style={{ display: 'flex', width: '100%', justifyContent: 'center' }}>
          <Button onClick={increment} style={{ margin: '5px' }}>Aumentar</Button>
          <Button onClick={decrement} style={{ margin: '5px' }}>Diminuir</Button>
          <Button onClick={reset} style={{ margin: '5px' }}>Resetar</Button>
        </div>
        <Progress current={value} showLabel />
      </>
    );

} }

</Playground>

### Progresso de Tarefa

O propósito principal do componente é mostrar o progresso de uma tarefa.   Observe a simulação do comportamento do componente.

<Playground>
{
  () => {
    const [value, setValue] = React.useState(0);

    timer = useTimer = () => {
      setTimeout(() => {
        setValue(prev => (prev < 100) ? prev + 1 : prev);
        useTimer();
      }, Math.floor(Math.random() * 2000 + 700));
    };
    useTimer();

    return (
      <>
        <div style={{ display: 'flex', width: '100%', justifyContent: 'center' }}>
          <Button onClick={() => setValue(0)} style={{ margin: '5px' }}>Resetar</Button>
        </div>
        <Progress current={value} showLabel />
      </>
    );

} }

</Playground>
